Quickstart - package.json

{
    "main": "index.html",
    "name": "nw-demo",
    "description": "demo app of node-webkit",
    "version": "0.1.0",
    "keywords": [ "demo", "node-webkit" ],
    "window": {
        "title": "node-webkit demo",
        "toolbar": true,
        "frame": false,
        "width": 800,
        "height": 600,
        "position": "mouse",
        "min_width": 400,
        "min_height": 200,
        "max_width": 800,
        "max_height": 600
    },
    "webkit": {
        "plugin": true
    }
}

package.json

  • main - this is the launch HTML page.
  • name - this is the name
  • window - object (to be defined in next 2 slides)
  • webkit - object (to be defined in 3 slides)
  • chromium-args - command line arguments
  • js-flags - arguments passed to JS engine
  • inject-js-start/inject-js-end

package.json - windows

  • title - (string) default window title
  • width/height - (int) intial height & widths
  • toolbar - (boolean) -showing toolbar
  • icon - path to window's icon
  • position - (null, center, mouse) where window will be placed
  • min_width/max_width/min_height/max_height
  • as_desktop - (boolean) show desktop background under x11

package.json - windows cont

  • resizable - (boolean)
  • always-on-top - (int) top of other windows
  • fullscreen - (boolean) available or not
  • show_in_taskbar - (boolean) shown in taskbar or dock
  • frame - (boolean) make window frameless
  • show - (boolean) hidden on startup
  • kiosk - (true/false) full screen and prevent from leaving app

package.json - webkit

  • plugin - load external browser plugins like Flash.
  • java - load applets
  • page-cache - to enable page cache

package.json - other fields

  • description - this is the launch HTML page.
  • version - this is the name
  • keywords - object (to be defined in next slide)
  • maintainers/contributers
  • bugs - url for submitting bugs can be http or mailto
  • licenses
  • repositories

Build OSX

npm install node-webkit-builder

Build Windows

copy /b nw.exe+app.nw app.exe
Ship nw.pak and icudt.dll with nw.exe

Build Linux

cat /usr/bin/nw app.nw > app && chmod +x app 
Ship nw and libffmpeg.so

API - Window

https://github.com/rogerwang/node-webkit/wiki/Window

  • open(url) - opens a URL
  • showDevTools shows the dev tools
  • Events - document-start, loading, loaded, etc.

API - Menus

https://github.com/rogerwang/node-webkit/wiki/Menu

  • Menu.createMacBuiltin(appname, options) - adds default menus to app
  • Menu.items[x].click - attaches event listener to this menu item
  • Menu.popup - pop-up menu

API - Platform Services

  • App - Proxy Configuration, crashDump, Origin Policy, argv (command line args)
  • Clipboard - Cut/Paste
  • Tray - Status Item, Status Icon, and System Tray
  • File Dialogs - More than <input type="file">
  • Shell - Desktop related jobs.
  • Handling files and arguments
  • Screen - OS Level Screen Events

Storage

  • Web SQL - Deprecated but it still works
  • IndexDB - NOSQL successory to WebSQL
  • PouchDB - Embedded inspired by CouchDB
  • EJDB - Embedded JSON Data Engine
  • NeDB - Node Embedded Database
  • StoreDB - based on localStorage
  • Web Storage - localStorage, sessionStorage
